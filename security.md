# Security Policy

## 🔒 Supported Versions

We support the latest version of **Ethical Hacking Guide & Code Simulator**. Please ensure you are using the latest release for optimal security.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Fully supported |
| < 1.0   | ❌ No longer supported |

## 🛡️ Reporting a Vulnerability

If you identify a security issue, please follow responsible disclosure and report it directly to [security@example.com](mailto:security@example.com). Include a description, steps to reproduce, and any additional information that will help us resolve the issue.

## 🔐 Security Best Practices

To maintain security while using this guide and simulator, please:

- **Use in a Secure Environment**: Avoid running the simulator on sensitive networks.
- **Follow Legal Guidelines**: Only use the simulator in a lawful, ethical manner.
